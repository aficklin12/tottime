ABC Quality Standards

Element I: Program Administration and Structure

Indicator Description Points Documentation
I.A The program utilizes staff and parent program evaluations to inform written plans for continuous quality improvement. 3 pts
I.B.1 The program has a transition policy that includes practices to support successful internal transitions and external transitions. 2 pts
I.B.2 The program has a written Inclusion Policy. Up to 4 pts

Element II: Staff Education and Professional Development

Indicator Description Points Documentation
II.A Program Leadership/Administrators meet education qualifications. Up to 12 pts
II.B Teachers meet education qualifications. Up to 12 pts
II.C.1 Program leadership provides early care and education professionals with an annual written evaluation based on supervision and observation of performance, which includes feedback on strengths and areas that need improvement. Up to 3 pts

Physical Activity

Indicator Description Points Documentation
III.B.1 The program's physical activity policy and practices encourage the well-being of children. 3 pts

Healthy Social-Emotional Development (Mental Health)

Indicator Description Points Documentation
III.C.1 The program's discipline policy is founded in an understanding of social-emotional development/behavioral health and the program staff collaborate with parents to plan developmentally appropriate and evidence-based strategies to support children with challenging behaviors. 4 pts

Additional Nutrition Requirements

Indicator Description Points Documentation
III.A.2 The program develops and maintains menus that reflect nutritionally, well-balanced healthy meals and/or snacks for all children. 3 pts

Child Screenings and Referrals

Indicator Description Points Documentation
III.D.1 The program conducts developmental, vision and hearing screenings and shares information about a child's growth and progress, based on results of the screenings, with families confidentially according to the instruments' implementation timeframe. 6 pts
III.D.2 The program partners with families to make appropriate referrals based on parental concerns/requests, child screening, and/or on-going documentation. 4 pts
III.D.3 The program collaborates with experts, professionals, and community resources/agencies/organizations to support children and families regarding medical, developmental, mental health, and/or other needs. 2 pts

Element IV: Family Communication and Support

Indicator Description Points Documentation
IV.A.1 The program utilizes a method to collect information about the child and his/her family. 3 pts
IV.A.2 The program communicates with families in multiple ways, including a plan or policy for daily communication between families and teachers. 2 pts
IV.A.3 The program maintains a resource list and/or literature from appropriate services/agencies to support children and families. The resource list and/or literature includes appropriate mental health services/agencies. Up to 3 pts
IV.A.4 Teachers offer a minimum of 2 family/teacher conferences annually. 3 pts
IV.A.5 The program demonstrates multiple methods to support family engagement including opportunities to engage families based on cultural needs and interests. 3 pts
IV.A.6 The program has policies and/or practices in place to support the needs of dual language learners (DLL). 3 pts
IV.A.7 The program supports families by having a plan to communicate in their home language and providing policies and documents in their home language. 3 pts
